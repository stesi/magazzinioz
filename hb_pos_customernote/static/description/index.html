Pos Customer Note

This module allows the user to create a Customer Note on order lines, from POS Interface, to be shown on receipt and invoice.

Version

v14.0.1.0.0 (23 March 2022)
  • Initial Release

Go to Point of Sale > Configuration > Point of Sale and make sure to check the boolean "Customer Note"

Customer Note Button will be shown on the POS interface.

Click on the order line and then click on the "Customer Note" button.

The note is shown on the order line selected.

It is shown on the receipt.

It is shown also on the invoice.
